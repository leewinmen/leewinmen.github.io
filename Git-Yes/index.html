所需安装程序

  1. Bit Bash & Git GUI
  2. GitHubDesktop

开始操作

  1. cd desktop/test

    [^本地创建一个文件夹存放git，打开]:

  2. git init

    [^初始化命令，自行输入帐号密码]:

  3. git clone https://github.com/leewinmen/liweimin.github.io

    [^克隆仓库到本地]:

上传文件到仓库

  1. git add .

    [^添加命令：需要先把需要上传的文件放到clone下来的文件夹对应位置]:

  2. git commit -m"init project"

    [^确认命令]:

删除仓库中的文件

  1. cd xxx.github.io

    [^进入克隆下了的仓库文件夹，需要删除文件的目录下]:

  2. git rm -f index.html

    [^这里我需要删除index.html这个文件]:

    • git rm -r index

    • [^这是删除文件夹的命令，比如这里删除index文件夹，替上面第二步]:

  3. git commit -m 'index.html'

    [^对这个文件前面的操作进行却]:

  4. git remote add origin https://github.com/leewinmen/liweimin.github.io.git

  5. git push -u origin master

    [^这样就把仓库里的某个文件删除了]:

    • git push https://github.com/leewinmen/liweimin.github.io
